# text_summarizer
text_summarizer

1. git clone
    ```sh
    git clone https://github.com/mr-noww/text_summarizer.git
    cd text_summarizer
    ```
2. Create and activate virtual env
    ```sh
    python -m venv venv
    .\venv\Scripts\activate
    ```
3. Install Dependencies
    ```sh
    pip install -r requirements.txt
    ```
4. Run Project
    ```sh
    python text_summarizer.py
    ```
